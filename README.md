Для запуска проекта необходимо установить зависимости при помощи "npm install", затем зпустить проект командой "npm run dev", после этого в браузере открыть "http://localhost:5173"
Использованные библиотеки: react-redux, @reduxjs/toolkit, @tanstack/react-table, sass-embedded
Архитектура: в папке api находятся файлы, используемые в сетевом взаимодейстаии, в папке components файлы компонентов, в папке store файлы, связанные с redux
